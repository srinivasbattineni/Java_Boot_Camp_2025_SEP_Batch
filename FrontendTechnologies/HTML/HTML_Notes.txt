Attribute			Description
----------------------------------------------------
id			Unique identifier for an element
class			Assigns one or more class names (used with CSS and JavaScript)

title			Tooltip text (shows when you hover over the element)
href			Specifies the URL for a link (<a>)
src			Specifies the path to an image or media file (<img>, <script>, etc.)
alt			Alternative text for images if image fails to load
type			Specifies the type of input, button, or script
value			Defines the initial value of an input element
